> **Gung12** — DAST specialized in web forms. Detects **12 OWASP categories** (SQLi, XSS, SSTI, XXE, NoSQLi, CSRF…), supports **SPAs** (Angular/React/Vue) via Playwright, optional **AI-assisted triage** (Gemini/Groq), WAF-bypass mode, and CI/CD-friendly exit codes. Cross-platform binaries in [Releases](https://github.com/DenReanin/gung12/releases).
